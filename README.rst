libxmos-avb-linux
=================

A port of the xmos sw_avb firmware from xcore to linux:
https://github.com/xcore/sw_avb
(https://github.com/xcore/sc_avb)

Warning! This repository is in its early stages.
Most of the code will not compile or behave as expected.

The end goal of this project is to provide AVB support to
linux devices using standard code from XMOS.
